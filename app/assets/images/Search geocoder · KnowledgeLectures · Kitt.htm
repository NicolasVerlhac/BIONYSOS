Geocoding

Address -> GPS Coordinates

"Eiffel Tower" → { lat: 48.8582, lng: 2.2945 }

Start with the Rails Minimal template

rails new \
  --webpack \
  --database postgresql \
  -m https://raw.githubusercontent.com/lewagon/rails-templates/master/minimal.rb \
  geocoding

cd geocoding

A new gem to your Gemfile

Let’s add the geocoder gem.

# Gemfile
gem 'geocoder'

bundle install
rails generate geocoder:config

Some configuration first

Open config/initializers/geocoder.rb:

Geocoder.configure(
  # [...]
  units: :km, # defaults to miles (:mi)
  # [...]
)

We will be using nominatim as geocoding provider, which uses OpenStreetMap’s data. This is limited to 1 request per second.

Suppose we already have a Flat

# for teacher's demo only:
rails g scaffold Flat name address

The address is just a string, like "16 Villa Gaudelet, Paris".

Add columns to your model

# app/models/flat.rb
class Flat < ApplicationRecord
end

Add two columns to your model you want to geocode:

(Replace Flat with the model you want to geocode!)

rails g migration AddCoordinatesTo*Flats* latitude:float longitude:float

rails db:migrate

Geocoding

Suppose that you store the address in an address column:

# app/models/flat.rb
class Flat < ApplicationRecord
  geocoded_by :address
  after_validation :geocode, if: :will_save_change_to_address?
end

Result

rails c

Flat.create(address: '16 Villa Gaudelet, Paris', name: 'Le Wagon HQ')
# => #<Flat:0x007fad2e720898
#  [...]
#  name: "Le Wagon HQ",
#  latitude: 48.8648482,
#  longitude: 2.3798534,
#  address: "16 Villa Gaudelet, Paris">

Search

Flat.near('Tour Eiffel', 10)      # venues within 10 km of Tour Eiffel
Flat.near([40.71, 100.23], 20)    # venues within 20 km of a point

Mapbox Maps

We need to set up an API key for Mapbox

Visit Mapbox and create an account.




On your account page you will see a section titled
Access Tokens. Either use your default token, or create a new token for your project.

Add this key to your .env.

# .env
MAPBOX_API_KEY=pk.eyJ1IjoicGR1b****************yZvNpTR_kk1kKqQ

Security

You must monitor the use of your key on your Mapbox account. For maximum security, refresh your token periodically so that nobody else can use your token!

Controller

You want to display a map on the flats#index route:

# app/controllers/flats_controller.rb
class FlatsController < ApplicationController
  def index
    @flats = Flat.geocoded #returns flats with coordinates

    @markers = @flats.map do |flat|
      {
        lat: flat.latitude,
        lng: flat.longitude
      }
    end
  end
end

View

<!-- app/views/flats/index.html.erb -->

<div
  id="map"
  style="width: 100%;
  height: 600px;"
  data-markers="<%= @markers.to_json %>"
  data-mapbox-api-key="<%= ENV['MAPBOX_API_KEY'] %>"
></div>

yarn add mapbox-gl

// app/javascript/plugins/init_mapbox.js
import mapboxgl from 'mapbox-gl';

const initMapbox = () => {
  const mapElement = document.getElementById('map');

  if (mapElement) { // only build a map if there's a div#map to inject into
    mapboxgl.accessToken = mapElement.dataset.mapboxApiKey;
    const map = new mapboxgl.Map({
      container: 'map',
      style: 'mapbox://styles/mapbox/streets-v10'
    });
  }
};

export { initMapbox };

Fit map to markers

const fitMapToMarkers = (map, markers) => {
  const bounds = new mapboxgl.LngLatBounds();
  markers.forEach(marker => bounds.extend([ marker.lng, marker.lat ]));
  map.fitBounds(bounds, { padding: 70, maxZoom: 15, duration: 0 });
};

if (mapElement) {
  // [ ... ]
  fitMapToMarkers(map, markers);
}

Going Further

Info Windows (1)

First, lets update our controller to pass more info about our flats in our view:

# app/controllers/flats_controller.rb
class FlatsController < ApplicationController
  def index
    @flats = Flat.geocoded

    @markers = @flats.map do |flat|
      {
        lat: flat.latitude,
        lng: flat.longitude,
        infoWindow: render_to_string(partial: "info_window", locals: { flat: flat })
      }
    end
  end
end

Custom Markers (2)

markers.forEach((marker) => {

  const popup = new mapboxgl.Popup().setHTML(marker.infoWindow);

  // Create a HTML element for your custom marker
  const element = document.createElement('div');
  element.className = 'marker';
  element.style.backgroundImage = `url('${marker.image_url}')`;
  element.style.backgroundSize = 'contain';
  element.style.width = '25px';
  element.style.height = '25px';

  // Pass the element as an argument to the new marker
  new mapboxgl.Marker(element)
    .setLngLat([marker.lng, marker.lat])
    .setPopup(popup)
    .addTo(map);
});

Address input autocomplete

We will use a service called Algolia Places which also uses OpenStreetMap

yarn add places.js
touch app/javascript/plugins/init_autocomplete.js

// app/javascript/plugins/init_autocomplete.js
import places from 'places.js';

const initAutocomplete = () => {
  const addressInput = document.getElementById('flat_address');
  if (addressInput) {
    places({ container: addressInput });
  }
};

export { initAutocomplete };

Styling your map

You can use the Mapbox Studio to design your own maps, or use existing designs

Then just update the style key, when initialising your map:

const map = new mapboxgl.Map({
  container: 'map',
  style: 'mapbox://styles/pdunleav/cjofefl7u3j3e2sp0ylex3cyb' // <-- use your own!
});
